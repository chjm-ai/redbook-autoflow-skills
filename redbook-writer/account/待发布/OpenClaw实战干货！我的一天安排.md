很多人问我：OpenClaw 到底能干啥？

今天把我一天的使用场景全拆解给你们，适合打工人直接抄作业。

📌 晨间例行（7:00-9:00）

设置好的自动化：
1. 晨报推送 → 聚合科技/产品/设计圈重要信息
2. 待办同步 → 从 Things + 邮件里提取今天要做的事
3. 日程提醒 → 提前把会议链接、相关材料准备好

💡 关键配置：用 RSS + AI 摘要，只看你关心的领域，不会被信息淹没。

📌 工作时段（9:00-12:00）

高频使用场景：
1. 邮件速读 → AI 总结每个任务核心诉求，标注优先级
2. 快速回复 → 常见询问用模板，复杂的我改一下发
3. 会议辅助 → 录音+实时转写+自动提取 action items

💡 关键配置：把公司常用话术存成快捷指令，回复效率翻倍。

📌 创作时段（14:00-18:00）

深度工作场景：
1. 选题灵感 → 让 AI 分析近期热点，结合我的定位给选题建议
2. 资料收集 → 指定几个信源，自动抓取关键信息做对比表
3. 内容生成 → 我给框架，AI 写初稿，我负责改和加个人风格
4. 视觉产出 → 描述画面，AI 生成配图/封面

💡 关键配置：建立自己的 prompt 库，不同场景用不同模板。

📌 晚间学习（21:00-22:00）

持续成长场景：
1. 文章精读 → 把长文丢给 AI，输出结构化笔记
2. 知识归档 → 自动分类存到 Obsidian，双向链接建好
3. 复习提醒 → 按艾宾浩斯曲线推送之前学过的内容

💡 关键配置：Obsidian + OpenClaw 联动，形成知识闭环。

📌 核心心法

AI 不是替代你思考，是帮你处理「查找、整理、格式化」这些体力活。真正的判断和创意还是要人来。

用好 OpenClaw 的关键是：清楚自己要什么，然后让它去执行。
